# 3D Geometry Editor - Koteika Edition

Интерактивный 3D-редактор для создания и редактирования геометрических объектов: кубы, сферы, пирамиды, икосаэдры. Реализован на Python с использованием библиотеки Dear PyGui

## Возможности

- **Создание 3D-объектов**: куб, сфера (треугольная/четырёхугольная), пирамида, икосаэдр
- **Редактирование**: перемещение точек мышью или точный ввод координат
- **Управление камерой**: вращение, панорамирование, масштабирование
- **Построение сечений**: произвольной плоскостью с визуализацией
- **Управление видимостью**: отсечение задних граней, рёбер и точек
- **Сохранение/загрузка**: JSON-сериализация всей сцены
- **Экспорт изображений**: сохранение вида в PNG
- **Настройка цветов**: индивидуальная настройка цветов граней, рёбер, точек и сечений

## Требования

- Python ≥ 3.8
- Windows / Linux / macOS

# Управление

## Камера

- Левая кнопка мыши + движение — вращение сцены
- Shift + левая кнопка + движение — панорамирование
- Колёсико мыши — масштабирование

## Редактирование

- Клик левой кнопкой по точке — выбор точки
- Ctrl + перетаскивание выбранной точки — перемещение точки мышью
- Ввод координат + Apply Point Position — точное перемещение

## Создание объектов

- Add Cube — создать куб
- Add Pyramid — создать пирамиду
- Add Sphere — создать сферу (треугольники)
- Add Sphere quad — создать сферу (четырехугольники)
- Add Icosahedron — создать икосаэдр
- Clear Scene — очистить сцену

## Сечения

- Show section — включить/выключить сечение
- Nx, Ny, Nz — нормаль плоскости
- Offset — смещение плоскости

## Сохранение и экспорт

- Save Scene — сохранить сцену в JSON
- Load Scene — загрузить сцену из JSON
- Export PNG — экспортировать текущий вид в PNG
